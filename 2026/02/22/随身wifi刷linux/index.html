随身wifi刷linux

backup中是所有分区的备份，不需要下载，在过程中需要你自己备份自己手头的随身wifi

1、看芯片

​ 拆下全部屏蔽罩，查找最大的一块芯片，看芯片上方丝印，一般来说会有以下几种情况：

  • QUALCOMM MSM8916：高通410，也是玩法最多的；

  • QUALCOMM MSM8909：高通210，玩法仅次于410；

    本人的为高通410，板子为boot-ufi001c

2、安装 9008 驱动

​ 运行9008driver.exe

3、安装 ADB 工具

​ 解压UFI003.zip，并进入用CMD解压文件夹，可直接使用adb不需要添加环境变量

​ 打开命令提示符，执行 adb version。有如下输出，则表明配置正确。

4、安装 Miko 备份工具

​ 解压MiKo_Loder Pro.zip，点击里面的miko.exe，安装完之后进入安装目录，将loader.exe拖进去，之后使用Miko工具均从loader.exe打开

随身wifi进入9008``模式，之后执行下图步骤

​ 若需要恢复则：随身wifi进入9008模式，执行下图步骤

5、开启 ADB

​ 将随身 WiFi 插入电脑 USB 接口。打开设备管理器，如果此时多了一个名叫 Android Composite ADB Interface 的设备，则表明随身 WiFi 已经开启 ADB，无需执行其他操作。

​ 如果没有出现这个设备，则需要手动开启 ADB。以 UZ801 型号为例，连上随身 WiFi 的热点，并访问 http://192.168.100.1/usbdebug.html 即可开启 ADB，再次访问即可关闭。

​ 部分 UFI003 型号的随身 WiFi 默认不开 ADB，需要手动进 9008 模式刷入已开启 ADB 的安卓固件。

固件下载地址：https://www.123pan.com/s/NUK8Vv-nRdl 提取码：1LEW

​ 手动进 9008 模式的方法：按住 reset 键不放（随身 WiFi 背面二维码右上角的小孔，用取卡针或牙签戳即可），并将随身 WiFi 插入电脑，松开 reset 键。

5、备份原厂固件

​ 打开命令提示符，执行 adb reboot edl，此时设备管理器中会多出高通 9008 设备。

6、刷入系统

​ 之后按照下图进行替换about，之后在弹出的页面选择你所下载的emmc_appsboot-test-signed.mbn

​ 当左侧提示aboutDone之后，进行格式化boot

​ 之后就可以重启了，重启之后执行adb reboot bootloader进入fastboot模式，就可以进行刷入系统了，在OpenStick内下载系统了，下载完成后运行压缩包内的flash.bat就可以刷入了，期间点几个回车

安装 RNDIS 网络驱动

​ 重新插入随身 WiFI，打开任务管理器，在其他设备中会出现一个未知设备或以 RNDIS 开头的设备

如果设备管理器只显示 Android Device，右键更新驱动程序，依次选择更新驱动程序–>浏览我的电脑以查找驱动程序–>让我从计算机上的可用驱动程序列表中选取->去掉打勾显示兼容硬件->左侧找到Microsoft，右侧找到USB移动宽带网络适配器->安装，然后重新插入随身 WiFi，会出现一个未知设备，再根据下述步骤进行操作。

​ 右键，更新驱动程序–>浏览我的电脑以查找驱动程序–>让我从计算机上的可用驱动程序列表中选取–>显示所有设备，等待列表加载完成。

​ 左侧选择 Microsoft，右侧选择 基于远程 NDIS 的 Internet 共享设备，在弹出窗口中选择是，并确定。

​ 至此，RNDIS 驱动安装成功。

登录debian

​ 驱动安装成功后，电脑上会多出一个名叫 4G_UFI_123456 的 NDIS 设备，同时会出现一个同名的热点，密码为 12345678。

​ Debian 默认用户名为 user，密码为 1，root 密码也为 1。请务必及时修改密码（命令：passwd）。

随身 WiFi 联网

​ 此时随身 WiFi 并没有联网，必须要关闭热点才能连接 WiFi。下面介绍详细步骤。

常用命令

命令 所属目录 主要功能 常用示例
lp /usr/bin 提交打印任务（最常用） lp -d 打印机名 文件.pdf
lpr /usr/bin 另一种提交打印任务的命令 lpr -P 打印机名 文件.pdf
lpq /usr/bin 查看打印队列（查看等待打印的任务） lpq -a（查看所有队列）
lprm /usr/bin 取消打印任务 lprm 任务ID 或 lprm -（取消所有）
lpstat /usr/bin 显示打印机状态和任务信息 lpstat -p -d（查看打印机）
lpoptions /usr/bin 设置或查看打印选项 lpoptions -d 打印机名（设默认）
lpunlock /usr/bin 解锁被锁定的打印机队列 lpunlock 打印机名
lpadmin /usr/sbin 添加、删除或配置打印机（需root） lpadmin -p 打印机名 -v 设备
lpinfo /usr/sbin 列出可用的打印机驱动和接口 lpinfo -v（查看可用连接）
lpmove /usr/sbin 将任务从一个队列移到另一个 lpmove 任务ID 目标打印机
lpc /usr/sbin 控制打印机状态（启动/停止/启用） lpc status（查看状态）

在 lp 命令中，复杂的打印选项主要通过 -o 参数来指定。根据 CUPS 的官方文档，以下是常用的复杂打印选项 ：

页面布局与排版选项

选项 说明 示例值
-o number-up=N 每张纸上打印N页 2, 4, 6, 9, 16
-o page-border=border 每页添加边框 single（单线）, double（双线）, single-thick（粗单线）
-o number-up-layout=layout 多页排列顺序 lrtb（从左到右，从上到下）, tblr（从上到下，从左到右）等
-o page-set=set 打印特定页面集 odd（奇数页）, even（偶数页）, booklet（小册子）
-o outputorder=reverse 逆序打印 reverse
-o mirror 镜像打印 （无值，直接使用）

双面与方向选项

选项 说明 示例值
-o sides=option 单面/双面设置 one-sided（单面）, two-sided-long-edge（双面长边）, two-sided-short-edge（双面短边）
-o orientation-requested=N 打印方向 3（纵向）, 4（横向逆时针）, 5（横向顺时针）, 6（反向纵向）
-o landscape 横向打印（简化版） （无值，直接使用）

选项 说明 示例值

-o print-quality=N 打印质量 3（草稿）, 4（标准）, 5（最佳）
